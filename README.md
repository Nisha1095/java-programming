# java-programming
This repository contains my daily progress, tasks, and code implementations from a 10-day Java programming training program conducted by oracle.
The training is designed to build a strong foundation in Java programming, starting from basics to intermediate concepts.
Topics Covered:
🔹 Day 1: Introduction to Java including
          History and features of Java,
          JDK, JRE, JVM,
          Setting up Java environment,
          First Java program.
🔹 Day 2: Variables,Data Types,Operators and Expressions.
🔹 Day 3: Control Statements and Loops.
🔹 Day 4: Input types in Java and String class.
🔹 Day 5: Methods using Java.
🔹 Day 6: Arrays and ArrayList in Java with in built functions in it.
